# 🌾 DirectFarm

DirectFarm is a web application that connects farmers directly with consumers and retailers, removing middlemen. It allows farmers to list products, set prices, and enables users to buy fresh produce easily.

---

## 🚀 Features

* 👨‍🌾 Farmer product listing
* 🛒 Customer product browsing
* 📦 Basic order/cart system
* 🖼️ Product images support
* 🔐 Simple login system

---

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript
* Firebase

---

## ⚙️ Getting Started

### 1. Clone Repository

```bash
git clone https://github.com/sujay-0-dev/Direct-Farm.git
cd Direct-Farm
```

### 2. Run Project

Just open `index.html` in your browser
(or use Live Server in VS Code)
